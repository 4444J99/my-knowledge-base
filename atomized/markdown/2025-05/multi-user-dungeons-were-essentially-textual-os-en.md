Another example is TempleOS, an experimental OS created by Terry A. Davis, which is a biblical-themed operating system with an interface and language (HolyC) influenced by religious symbolism . TempleOS, while idiosyncratic, demonstrates how an operating environment can be built around a personal symbolic ontology (in his case, a dialog with the divine). For MythOS, drawing on mythic structure could be as simple as naming the top-level categories after archetypal domains (e.g. Yggdrasil for a root that branches into knowledge), or as elaborate as a gamified world where each task or file is part of an epic quest storyline. The benefit of symbolic interfaces is that they create emotional and cognitive resonance – your data is not in abstract folders but in places and narratives that mean something to you.
