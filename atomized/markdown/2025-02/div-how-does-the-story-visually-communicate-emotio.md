<div>2. Vomit Drafting as an Umbrella Mindset</div>
<div><br></div>
<div>The Vomit Draft is a philosophy, not an activity—it’s the permission to write without judgment. The idea is:</div>
<div>  •  No one will see this but you.</div>
<div>  •  It’s supposed to be messy, ugly, and bad.</div>
<div>  •  This is where you find the raw, unpolished ideas.</div>
<div>  •  Every writer starts with shit before refining it.</div>
<div><br></div>
<div>Activity: Give Yourself Permission to Write Badly</div>
<div>  •  Set a word count goal, not a quality goal (e.g., “Write 500 words, no matter what”).</div>
<div>  •  Use “I don’t know” placeholders (e.g., “Something happens here that I’ll figure out later”).</div>
<div>  •  Accept that exploratory writing isn’t the final product—it includes:</div>
<div>  •  Character bios</div>
<div>  •  Theme exploration</div>
<div>  •  Worldbuilding notes</div>
<div>  •  Random dialogue ideas</div>
