# <div>- Students write a description of their favorite food using all five senses

## Context
From document: **ENG 101 - Week 2 Class Schedules (Days 5, 6, 7, & 8)** — chunk 16 (chunk 16/31)

## Content
<div>- Students write a description of their favorite food using all five senses **without naming the food**.</div>
<div>- Peers read each other’s descriptions and try to guess the food.</div>
<div>- The class discusses what worked well and how descriptions could be improved.</div>
<div><br></div>
<div>📌 **Key Takeaways:**  </div>
<div>- Sensory writing makes experiences feel real.</div>
<div>- Concrete details create stronger emotional connections with readers.</div>
<div>- Word choice and structure impact the effectiveness of description.</div>
