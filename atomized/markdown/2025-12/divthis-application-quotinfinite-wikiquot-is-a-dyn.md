## Content
<div>This application, &quotInfinite Wiki,&quot is a dynamic, AI-powered hypertext encyclopedia. Its core functionality allows users to explore a web of interconnected knowledge where every word in a definition becomes a hyperlink to a new, dynamically generated entry.</div>
<div><br></div>
<div>Here is a breakdown of its functionality:</div>
<div><br></div>
<div>Core Concept</div>
<div>The application is built on the idea of a self-generating wiki. When a user looks up a topic, the app doesn't pull from a database. Instead, it uses the Google Gemini API to generate a concise, encyclopedia-style definition in real-time. The key feature is that once the definition is displayed, every single word within it becomes a clickable link, allowing the user to instantly navigate to a new definition for that word, creating an endless &quotrabbit hole&quot of exploration.</div>
<div><br></div>
<div>Key Features &amp User Flow</div>
<div>Topic Exploration:</div>
<div>Search: Users can enter any word or concept into a search bar to get its definition.</div>
<div>Random Discovery: A &quotRandom&quot button provides a curated, interesting topic, encouraging serendipitous learning.</div>
<div>Hypertext Navigation: This is the central mechanic. Clicking any word in a generated article makes that word the new topic, seamlessly chaining from one concept to the next.</div>
