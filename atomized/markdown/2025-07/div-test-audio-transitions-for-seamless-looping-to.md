<div>3. Educational Journey in 3D</div>
<div>Approach:</div>
<div>  •  Timewalk Onboarding: Create a cinematic fly-through using Unity’s Timeline for each era, narrated by the Archivist (TTS or pre-recorded voice). Keep it under 30 seconds to avoid overwhelming new players.</div>
<div>  •  In-World Tutorials: Use 3D UI canvases for scene-anchored prompts. Highlight valid moves with glowing checkers and animated trails (via Unity’s Particle System). Offer skippable tutorials for returning players.</div>
<div>  •  Challenge Scenarios: Implement puzzles as scripted scenes with historical avatars (e.g., Roman Centurion). Use Unity’s Animator for avatar gestures and dialogue triggers.</div>
<div>  •  Cultural Variants: Package rule variants (Nard, Tapa) as modular scripts (ruleset.js compiled to C#). Display micro-tutorials as 3D pop-ups with ghost move animations.</div>
<div>  •  Philosophical Prompts: Trigger brief quotes (e.g., Stoic maxims) as 3D text overlays after key moves, with a “Learn More” button linking to lore.</div>
<div>  •  Progress Tracking: Store skill maps and badges in Firebase Firestore, syncing progress across devices.</div>
