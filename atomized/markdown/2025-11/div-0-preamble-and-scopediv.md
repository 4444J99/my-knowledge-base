<div>  0.  Preamble and Scope</div>
<div><br></div>
<div>This document is the master specification for a unified, self-evolving creative-operating-system. It defines conceptual layers, directory and naming structures, procedural generation flows, and automation patterns that turn individual AI runs into contributions to a single, long-lived corpus. It is written so a future AI, given only this text plus environment variables, can reconstruct the system and extend it.</div>
<div><br></div>
<div>All paths and identifiers here are conceptual and use environment-variable placeholders such as $OS_ROOT, $ARCHIVE_ROOT, $PROMPT_ROOT, $REPO_ROOT, $ORG_ID, and $AUTOMATION_ENGINE. Implementation details (scripts, frameworks, services) are intentionally abstracted.</div>
