## Content
<div>CREATING EVOLVING DRUM PATTERNS IN ULTRABEAT USING THE MIDI ENVIRONMENT</div>
<div><br></div>
<div>1. Create a ‘Software Instrument’ track.</div>
<div><br></div>
<div>2. Load ‘Ultrabeat’ on that track.</div>
<div><br></div>
<div>3. Duplicate that track 3 times, so that you have 4 Software Instrument tracks with Ultrabeat.</div>
<div><br></div>
<div>4. Create a different drum pattern on each of the 4 Ultrabeat instruments. Patterns that utilize many of the kit’s samples (busy patterns) work well.</div>
<div><br></div>
<div>5. Press ‘Command+0’ to open the ‘MIDI Environment’.</div>
<div><br></div>
<div>6. Go to the ‘Mixer layer’ of the ‘MIDI Environment’.</div>
<div><br></div>
<div>7. From the local ‘New’ menu in the MIDI Environment, create a ‘Monitor’ object.</div>
<div><br></div>
<div>Select the ‘Monitor’ object; in the ‘Object Inspector’ check the ‘Assignable’ checkbox.</div>
<div><br></div>
<div>8. From the local ‘New’ menu in the Environment, create an ‘Arpeggiator’ object.</div>
<div><br></div>
<div>9. Select the ‘Arpeggiator’ object and set the ‘Direction’, ‘Resolution’, and ‘Length’ parameters to ‘Random’ in the ‘Object Inspector’. After going through this process at least once, experiment with different Length and Resolution settings.</div>
