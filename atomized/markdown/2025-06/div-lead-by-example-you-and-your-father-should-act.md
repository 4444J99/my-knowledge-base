<div>System 2: Shared Notes for Equipment Requests</div>
<div>Goal: Create a centralized, easily accessible, and structured system for employees to request equipment and supplies, reducing back-and-forth trips.</div>
<div>Recommended Platforms: The key here is a system that allows for structured data entry, easy viewing, and perhaps even basic tracking.</div>
<div> * Google Sheets / Microsoft Excel (Shared Online):</div>
<div>   * Pros: Familiar to most, highly customizable, easy to set up.</div>
<div>   * Cons: Can get messy without clear guidelines, limited automation.</div>
<div> * Google Forms / Microsoft Forms + Google Sheet / Excel:</div>
<div>   * Pros: Enforces structured data entry, minimizes errors, automatically populates a spreadsheet.</div>
<div>   * Cons: Requires an extra step for the user to fill out a form.</div>
<div> * Simple Project Management Tool (e.g., Trello, Asana - Free/Low Cost Tiers):</div>
<div>   * Pros: Designed for task tracking, visual, can assign requests, set due dates.</div>
<div>   * Cons: Might be overkill initially, steeper learning curve for some.</div>
